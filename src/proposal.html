WDD 330 Final Project Proposal

Description

Overview
This website will be a quick and easy tool to look through upcoming anime titles, sorting by various parameters including show information and the presence or absence of user tags (probably a "star", "watching", and "dislike").
Target Audience
The target audience is regular anime watchers who are not "hardcore" enough to be easily on top of what's coming up and just want to keep up with what's available.
Major Functions
Major functions include listing anime titles by season, listing anime titles by ranking, sorting listed titles, marking titles with tags, and displaying additional details about the selected title.
External Data
I am using the myanimelist "Get seasonal anime" and "Get anime ranking" api endpoints.
Module List
  • AnimeList: Fetches and contains a list of anime titles.
  • Filter: Stores and manages filter settings.
  • AnimeDetails: Stores and displays the details of an anime in a modal.
